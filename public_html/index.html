ENS2001

Un ensamblador y simulador del estandar IEEE 694 con versiones en modo consola para DOS y GNU/Linux y en modo grafico para Windows.

Acerca de ENS2001

ENS2001 es una aplicación que integra un Ensamblador de un subconjunto de instrucciones del estándar IEEE 694 y un Simulador, capaz de ejecutar programas ensamblados para dicha implementación particular del estándar.

Se trata de una mejora sobre las herramientas disponibles hasta el momento, ASS y ENS96, a las que añade una arquitectura de máquina virtual mejorada, una implementación del lenguaje más homogénear y, principalmente, una nueva y cómoda interfaz gráfica.

La aplicación se ofrece en tres versiones alternativas, a saber: entorno gráfico para Microsoft Windows, línea de comando para MS-DOS y línea de comando para Linux.

Para usar la herramienta, simplemente hay que descomprimir en un directorio del disco duro los archivos contenidos en el paquete comprimido correspondiente a la versión que se quiera emplear. A la hora de ejecutar la aplicación, el directorio de trabajo no debe estar protegido contra escritura, ya que la herramienta genera archivos temporales durante la sesión.
